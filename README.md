#RecipeApp

Приложение для просмотра и сохранения рецептов блюд.
Пользователь может просматривать блюда, фильтровать рецепты по категориям, сохранять рецепты и рассчитывать порции.

##Основные возможности
- Категории блюд и выбор рецептов
- Страница с ингредиентами, их количеством и пропорциями
- Калькулятор порций
- Список избранных рецептов

##Технологии
- Kotlin
- Jetpack Compose
- MVVM, StateFlow
- Room Database
- Retrofit + OkHttp
- Hilt
